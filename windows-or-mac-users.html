What if You’re on Windows or Mac?

If you are not a Linux user - you still got options:

Windows Users

○ You can run this script inside Windows Subsystem for Linux (WSL).

  • Install WSL (Ubuntu is a great choice).
  • Once WSL is set up, open your Linux terminal and follow the same instructions as Linux users.
  • You can scan your Windows folders from WSL — just navigate to them (they’re usually under /mnt/c/Users/...).

Mac Users

This script might work on macOS since it has Bash and many of the required tools available via Homebrew.

⚠️ Be aware: These scripts haven’t been tested on macOS, so results may be unpredictable!

Better Way (for both system users)

If you don’t want to change your daily system:

  • You can boot into a Linux Live USB — no installation required, no changes to your existing OS.
  • Just run the script from the live session to check your files.
  • This gives you the benefits of Linux without switching full-time.
